PILARES DEL PENSAMIENTO COMPUTACIONAL
-
  1. descomposición: "dividir para conquistar" tomar un problema grande y descomponerlo en algo más pequeño y manejable.
  2. reconocimiento de patrones: "encontrar similitudes" observar tendencias dentro de un problema. si algo se repite o sigue una logica constante podemos autpmatizarlo.
  3. abrstracción: "lo importante vs el detalle" filtrar la información innecesaria y quedarse solo con las características que definen el problema. crear una representación simboloca del arealida.
  4. algoritmo: "la receta paso a paso" el diseño de una relgas ordenadas para resolver el problema. es el plan de acción que debe seguir el problema.
